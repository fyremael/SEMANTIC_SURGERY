# Reports

Generated reports are written to the reports directory.

- `css_probe_report.json` is the spec-shaped machine report.
- `css_probe_report.operator_cards.md` is the human-readable operator-card sidecar.
- `css_probe_results_presentation.html` is the static visual report.
- `css_probe_results_summary.svg` is a one-page visual summary.
- `css_probe_results_summary.png` is a preview image generated when Pillow is available.

Each probe entry includes a packet and certificate. The suite-level certificate records Phase 0 Level 1-6 coverage.
